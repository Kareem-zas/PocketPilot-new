# 🚀 PocketPilot

A smart personal finance management app built with Flutter (frontend) and Node.js/Express (backend).

## 📁 Project Structure

```
PocketPilot-new/
├── pockect_pilot/     # Flutter mobile app
└── backend/           # Node.js + Express + MongoDB API
```

## 🛠 Tech Stack

- **Frontend:** Flutter (Dart)
- **Backend:** Node.js, Express, MongoDB (Mongoose)
- **Auth:** JWT
- **AI:** Gemini API integration

## ✨ Features

- 📊 Expense & income tracking
- 🎯 Savings goals with progress tracking
- 📈 Stats & analytics charts
- 🤖 AI Pilot (Gemini-powered assistant)
- 🧾 Receipt scanner
- 🏅 Gamification (XP, levels, streaks)
- 🔔 Subscription radar
- 💵 Pocket Money tracker

## 🚀 Getting Started

### Backend
```bash
cd backend
npm install
npm run dev
```

### Flutter App
```bash
cd pockect_pilot
flutter pub get
flutter run
```
